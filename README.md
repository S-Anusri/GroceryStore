****Grocery Store Application****

Welcome to the Fresh Grover's Grocery Store Application Platform!

Grocery Store Application using HTML, CSS, JavaScript, Servlets and MySQL.

Completed the Application which enables functionalities for customers, sellers, enables transactions and generates reports.

Products can be viewed based on the product id, by category, and the user can filter the number of items to be displayed on the page at once (5, 10, etc.)

There is a provision where both customers and sellers can register themselves on the website to have access to various services. There is also a login page for further fine-grained control over updating products or to perform checkout at the cart, where the role of the person is determined during checkout and the appropriate pages / level of access control is provided accordingly.

The seller have the option to obtain detailed reports on sales during a particular month, between specific time intervals, etc.

And finally, a feedback form is provisioned where visitors (both registered and non-registered) can leave their review of the experience with the platform.
